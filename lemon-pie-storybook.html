Categorias de tokens

Os tokens seguem uma hierarquia semântica com 7 categorias: action (interações como botões e links), background (fundos de telas e containers), surface (cards, inputs, borders), text (tipografia), icon (ícones), decorative (gráficos e avatares). Cada categoria tem subcategorias por intenção (neutral, branded, positive, negative, warning, information) e intensidade (veryHigh → high → default → low → veryLow → onColor).

Instalação

Como integrar o Lemon Pie DS no seu projeto React Native.

1. Copiar os arquivos

# Copie as pastas tokens/ e components/ para o seu projeto cp -r lemon-pie-ds/tokens src/design-system/tokens cp -r lemon-pie-ds/components src/design-system/components

2. Envolver o app com o Provider

import { LemonPieThemeProvider } from './design-system/tokens'; export default function App() { return ( <LemonPieThemeProvider> <NavigationContainer> <RootStack /> </NavigationContainer> </LemonPieThemeProvider> ); }

3. Usar em qualquer componente

import { useTheme } from './design-system/tokens'; import { Button, Card } from './design-system/components'; function MyScreen() { const { colors, colorScheme, toggleColorScheme } = useTheme(); return ( <View style={{ backgroundColor: colors.background.neutral.default }}> <Card title="Bem-vindo" description="Lemon Pie DS"> <Button onPress={toggleColorScheme}> Trocar para {colorScheme === 'light' ? 'dark' : 'light'} </Button> </Card> </View> ); }

Dependências

Nenhuma dependência extra. Todos os componentes usam apenas React Native core (View, Text, Pressable, Modal, Animated, TextInput, ScrollView, Image, ActivityIndicator).
