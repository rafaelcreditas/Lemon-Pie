Cores Semânticas — Background

🎨 colorTheme

Tokens de cor para fundos de elementos. Organizados em 3 níveis de ênfase: low, default e high.

background/secondary

background/secondary/low
Fundo secundário sutil. Ideal para inputs, tabs e áreas que precisam de leve distinção do fundo.
FRAME_FILLSHAPE_FILL
background/secondary/default

FRAME_FILLSHAPE_FILL
background/secondary/high

Elevation / Sombras

🔢 numbers&effects

O token elevation/default é composto por 5 camadas de sombra (effects). Cada efeito tem uma variável de cor associada. Na implementação, use o Effect Style correspondente no Figma — não aplique as cores diretamente.

Token Descrição Escopo
elevation/default/effect 1/color Parâmetro X/Y/blur/spread/color do efeito elevation/default. Use o Effect Style correspondente. EFFECT_COLOR
elevation/default/effect 2/color 2 Parâmetro X/Y/blur/spread/color do efeito elevation/default. Use o Effect Style correspondente. EFFECT_COLOR
elevation/default/effect 3/color 3 Parâmetro X/Y/blur/spread/color do efeito elevation/default. Use o Effect Style correspondente. EFFECT_COLOR
elevation/default/effect 4/color 4 Parâmetro X/Y/blur/spread/color do efeito elevation/default. Use o Effect Style correspondente. EFFECT_COLOR
elevation/default/effect 5/color 5 Parâmetro X/Y/blur/spread/color do efeito elevation/default. Use o Effect Style correspondente. EFFECT_COLOR
⚠️ Nota para devs:

Não use estas variáveis de cor individuais como box-shadow separadas. Aplique o Effect Style elevation/default no Figma, que compõe todas as 5 camadas. Em código, implemente como um box-shadow multi-camada ou como um token composto.

Espaçamento

📏 spacing

Escala de espaçamento com 10 níveis (0–9). Cada nível tem uma descrição semântica de uso. Collection: 📏 spacing — escopo: GAP.

Token Nível Alias / Uso Visual
spacing/0 0px Reset — Sem espaçamento.
spacing/1 2px Micro — Separadores inline, dividers sutis.
spacing/2 4px Tight — Gap entre texto e ícone, items inline.
spacing/3 8px Base — Gap padrão entre elementos relacionados.
spacing/4 12px Base+ — Gap entre elementos com leve separação.
spacing/5 16px Group — Separação entre grupos de elementos.
spacing/6 20px Container — Padding interno de cards e containers.
spacing/7 24px Section — Padding de seções, margens de tela.
spacing/8 32px Block — Separação entre blocos de conteúdo.
spacing/9 48px Region — Espaçamento entre regiões de página.
